# ArcadePortfolio
concept for an arcade cabinate styled portfolio website 

## Build setup 
 ```
 # install dependances
 $ npm install 
 
 # serve application using fast refresh on port localhost:3000
 $ npm run dev
 
 # build app
 $ npm run build
 ```
 ## Tools
 - Typescript
 - SCSS
 - Next
